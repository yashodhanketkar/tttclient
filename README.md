Tictactoe client
================

This is source code for a tictactoe game, this serves as a frontend for the tictactoe server. This project is build using Reactjs and shadcn.

Tech Stack
----------

-	React.js
-	Shadcn + Tailwindcss

Installation
------------

1.	Clone the repository
2.	Create a `.env` file in the root directory of the project and add the following variables:

	```
	VITE_BASE_URL_HTTP=http://{{url}}/api
	VITE_BASE_URL_WS=ws://{{url}}
	```

3.	Run `pnpm install` to install dependencies

4.	Run `pnpm dev` to start the server

### Server

-	[tttserver](https://github.com/yashodhanketkar/tttserver)
